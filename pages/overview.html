About Gravity ChartsAbout Gravity Charts

Gravity Charts is a charting library built with D3 and React. It's designed as part of the Gravity UI ecosystem and provides powerful data visualization capabilities for modern web applications through simple configuration objects.

What you can already do with Gravity ChartsWhat you can already do with Gravity Charts

We have already implemented support for the main chart types:

  • Area
  • Bar-X
  • Bar-Y
  • Line
  • Pie
  • Scatter
  • Treemap (partial support)
  • Waterfall (partial support)
