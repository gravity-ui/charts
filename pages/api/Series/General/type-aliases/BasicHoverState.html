    Name Type Description
    brightness? number How much to brighten/darken the point on hover. Use positive to brighten, negative to darken. The behavior of this property is dependent on the implementing color space (more details). For example in case of using rgb color you can use floating point number from -5.0 to 5.0. Rgb color space is used by default. Default 0.3
    enabled? boolean Enable separate styles for the hovered series. Default true
