    category? string Corresponding value of axis category. Deprecated use x or y instead
    color? string Individual color for the data chunk (point in scatter, segment in pie, bar etc)

    opacity? number Individual opacity for the point.
    radius? number Individual radius for the point.
    x? string | number The x value of the point. Depending on the context , it may represents: - numeric value (for linear x axis) - timestamp value (for datetime x axis) - x axis category value (for category x axis). If the type is a string, then it is a category value itself. If the type is a number, then it is the index of an element in the array of categories described in xAxis.categories
    y? string | number The y value of the point. Depending on the context , it may represents: - numeric value (for linear y axis) - timestamp value (for datetime y axis) - y axis category value (for category y axis). If the type is a string, then it is a category value itself. If the type is a number, then it is the index of an element in the array of categories described in yAxis[0].categories
