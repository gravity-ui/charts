    Name Type Description
    label? string | number Data label value of the bar. If not specified, the x value is used.
    opacity? number Individual opacity for the bar.
    x? string | number The x value of the bar. Depending on the context , it may represents: - numeric value (for linear x axis) - timestamp value (for datetime x axis) - x axis category value (for category x axis). If the type is a string, then it is a category value itself. If the type is a number, then it is the index of an element in the array of categories described in xAxis.categories
    y? string | number The y value of the bar. Depending on the context , it may represents: - numeric value (for linear y axis) - timestamp value (for datetime y axis) - y axis category value (for category y axis).
